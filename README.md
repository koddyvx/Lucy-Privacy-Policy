Privacy Policy for Lucy Discord Bot
​Last Updated: [15/9/2025]

​1. Introduction
​This Privacy Policy outlines how Lucy ("the Bot") collects, uses, and protects information from its users and the Discord servers where it is active. By using Lucy, you agree to the terms described in this policy.

​2. Data We Collect
​To provide its services, Lucy may access and process the following data:
​User Information: User IDs, usernames, and discriminators to identify users for moderation and AI interaction.
​Message Content: As a bot with Message Content Intent, Lucy processes message text to identify prefix commands (e.g., for moderation and AI chat) and to filter prohibited content (Anti-link, Anti-nuke).
​Server Data: Server IDs and channel IDs to maintain settings and security configurations.

​3. How We Use Your Data
​The data collected is used strictly for the following purposes:
​AI Interaction: To process and respond to user queries via our AI system.
​Moderation & Security: To execute moderation commands, detect prohibited links, and provide Anti-nuke protection.
​Command Execution: To respond to prefix-based commands for server management.

​4. Data Storage and Retention
​Message Content: Lucy does not store the content of your private conversations or general chat messages on any permanent database. Messages are processed in real-time and discarded immediately after a command is executed or filtered.
​Server Settings: Custom configurations (like blacklisted words or moderation logs) are stored securely in our database and are deleted if the bot leaves the server.

​5. Data Sharing
​We do not sell, trade, or share any user or server data with third parties. Data is only used within the Discord environment to provide the Bot’s features.

​6. User Rights
​Users can request the removal of any stored settings related to them by contacting the Bot Developer or by removing the bot from their server, which triggers automatic data deletion for that guild.

​7. Contact Information
​If you have any questions regarding this Privacy Policy, please contact the developer via our Support Server or Discord DM.
